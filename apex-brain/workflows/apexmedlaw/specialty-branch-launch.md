# Specialty Branch Launch - ApexMedLaw

## Objective
Repeatable process for launching a new medical specialty branch under ApexMedLaw.

## Trigger
New specialty identified with at least 1 interested physician

## Steps
1. **Define** - Create entity config for new specialty (name, description, target case types)
2. **Recruit** - Source physicians via SEAK, LinkedIn, medical associations
3. **Pipeline** - Set up lead pipeline: which law firms handle this case type?
4. **Outreach** - Launch law firm outreach campaign for this specialty
5. **Branding** - Create sub-brand page under ApexMedLaw umbrella site
6. **CRM** - Add specialty to case management dashboard

## Skills Used
- email-outreach
- competitor-analysis
- browser-automation

## Current Branches
- Neurology/Neurosurgery (NLC, active)
- Critical Care (launching)
- GI (early stage)
