# Website Launch - Club Haus
schedule: one-time

## Objective
Build and deploy a website for Club Haus Barber Parlor with booking integration.

## Steps
1. **Design** - Create modern barbershop landing page matching brand (near GSU, premium feel)
2. **Pages** - Home, Services + Pricing, Book Now, About, Contact
3. **Booking** - Integrate Square Online scheduling widget
4. **Mobile** - Ensure mobile-first design (college students book on phones)
5. **Deploy** - Push to Vercel, connect custom domain
6. **SEO** - Basic local SEO (Google Business Profile, "barbershop near GSU" keywords)

## Skills Used
- frontend-design
- seo-content

## Tools Used
- tools/deploy_to_vercel.py
