# Physician Recruitment Pipeline - NLC
schedule: weekly

## Objective
Find and recruit neurologists, neurosurgeons, and other specialists to join the NLC/ApexMedLaw roster.

## Schedule
Weekly (Monday)

## Steps
1. **Source** - Search SEAK registry, LinkedIn, medical association directories for potential physicians
2. **Qualify** - Check credentials, specialty match, geographic coverage
3. **Outreach** - Draft personalized recruitment email/LinkedIn message
4. **Follow-up** - Automated 3-touch follow-up sequence (Day 1, Day 4, Day 10)
5. **Onboard** - When physician responds positively, send onboarding docs and add to CRM

## Skills Used
- email-outreach
- competitor-analysis

## Tools Used
- tools/send_email.py
- tools/browser_agent.py (for SEAK registry)

## Output
- New physician leads in CRM
- Outreach logs in outputs/nlc/recruitment/
