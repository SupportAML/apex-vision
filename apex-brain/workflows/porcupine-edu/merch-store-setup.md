# Merch Store Setup - Porcupine Edu
schedule: one-time

## Objective
Set up a customizable T-shirt store where visitors can buy porcupine T-shirts with their name printed on them.

## Steps
1. **Research** - Compare Printful vs Printify (pricing, quality, API, name customization support)
2. **Design** - Create 5-10 T-shirt designs featuring porcupine illustrations
3. **Integration** - Connect print-on-demand API to website (product pages, cart, checkout)
4. **Customization** - Build name input field that renders preview on T-shirt mockup
5. **Payments** - Stripe checkout integration
6. **Shipping** - Configure shipping rates via print-on-demand provider
7. **Launch** - Go live, test end-to-end purchase flow

## Recommendation
Printful is the better choice for name customization. Their API supports text personalization on products, has strong Next.js integration guides, and handles fulfillment/shipping automatically.
