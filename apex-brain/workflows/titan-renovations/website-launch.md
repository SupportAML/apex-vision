# Website Launch - Titan Renovations
schedule: one-time

## Objective
Build a professional website that generates inbound leads from property managers.

## Steps
1. **Design** - Clean, trustworthy landing page with before/after gallery
2. **Pages** - Home, Services, Gallery, Get a Quote, Contact
3. **Quote form** - Simple form: property type, damage description, photos upload, contact info
4. **Local SEO** - Target "property renovation Atlanta", "tenant moveout cleanup Atlanta"
5. **Google Business** - Set up and optimize Google Business Profile
6. **Deploy** - Push to Vercel

## Skills Used
- frontend-design
- seo-content

## Tools Used
- tools/deploy_to_vercel.py
