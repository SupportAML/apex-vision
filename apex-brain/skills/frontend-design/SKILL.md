# Frontend Design

Build websites, landing pages, and web applications.

## What It Does
- Create production-grade web pages using Next.js + Tailwind + shadcn/ui
- Design responsive layouts matching brand guidelines
- Build interactive web apps (not just static sites)
- Generate component-based, deployable code

## Use Cases
- NLC marketing website and case management dashboard
- Club Haus barbershop website with booking integration
- Titan Renovations website with quote request form
- A2Z Equity investor portal
- Internal tools and team whiteboards

## Inputs
- Entity name (loads brand.md)
- Page type (landing, dashboard, app, form)
- Feature requirements
- Reference designs (optional)

## Outputs
- Deployable Next.js code
- Responsive for mobile and desktop
- Saved to outputs/[entity]/web/

Available as superpowers plugin: frontend-design skill
